#### div p

选择div元素内部的所有p元素

#### div>p

选择父元素为div的所有p元素，与div p的区别在于，只会选择子元素，达不到孙子元素

#### div+p

选择紧接着div之后的p元素（**必须同级**）

#### div~p

在div之后的所有p元素（**必须同级**）

#### JS打印九九乘法表

```JavaScript
    <script>
        for (var i = 9; i >= 1; i--) {
            for (var j = 10 - i; j >= 1; j--) {
                document.write(i + "*" + (10 - j)+"="+(i*(10-j)))
                document.write("   ")
            }
            document.write("<br>")
        }
    </script>
```
